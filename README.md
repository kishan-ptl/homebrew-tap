# kishan-ptl/homebrew-tap

Homebrew tap for [Joystick](https://github.com/kishan-ptl/joystick) — mission
control for your terminal.

```sh
brew install --cask kishan-ptl/tap/joystick
```

Then open Joystick and run the one-time setup to wire the shell + Claude Code
hooks. Requires macOS 14+, Ghostty, zsh, jq, and Claude Code.
